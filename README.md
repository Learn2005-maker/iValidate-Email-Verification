# iValidate - Email Validator

A simple Email Validator web application built using HTML, CSS, and JavaScript.

## Features

* Validate email addresses using an Email Validation API
* Responsive design for desktop and mobile devices
* Loading animation while checking email validity
* Clean and user-friendly interface
* Displays detailed validation results

## Technologies Used

* HTML
* CSS
* JavaScript
* Fetch API


## How to Run

1. Clone the repository.
2. Open `index.html` in your browser.
3. Enter an email address.
4. Click **Submit** to validate the email.
